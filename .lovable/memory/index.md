Design system tokens and visual rules for Campaign Tracker dashboard

## Color System
- Primary accent: teal/cyan hsl(189 94% 37%) — #0891b2
- Secondary: #0e7490
- Hero gradient: teal-to-dark-teal
- Page bg: #f0f4f8 (light), #0F1117 (dark)
- Cards: white (light), #1A1D27 (dark)
- Sidebar: #1a3a4a teal-navy, 220px wide with text labels
- Success: #10b981, Warning: #f59e0b, Danger: #ef4444

## Terminology
- Revenue → LTV everywhere
- ARPU → LTV/Sub
- Ad Spend → Spend
- Set cost → Set Spend
- No Data → No Spend
- Model categories: Jess=Female, Zoey=Female, Mia=Trans, Flor=Female, Aylin=Trans

## Typography
- Font: system UI stack
- Labels: 11px uppercase tracking-wider muted
- Values: 13-24px bold
- Min text: 12px

## Components
- Cards: 16px radius, subtle shadow, card-hover class
- Badges: 8px radius
- Sidebar: full width 220px with icon+text nav items
- Status: SCALE=green, WATCH=amber, KILL/DEAD=red, NO SPEND=gray
